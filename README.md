## Tecnología

- HTML5, CSS3 (sin frameworks), JavaScript vanilla — sin build step ni dependencias `npm`.
- Fuentes: **Lora**, **DM Sans** y **Prata** vía Google Fonts.
- Smooth scroll con [Lenis](https://github.com/darkroomengineering/lenis) servido localmente.
- Tema claro/oscuro con persistencia en `localStorage`.
- i18n ES/EN con switch en el header.

## Ejecutar localmente

```bash
python3 -m http.server 8080
```

Y abrir <http://localhost:8080>.
